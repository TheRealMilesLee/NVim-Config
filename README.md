# NVim-Config  
A modular Neovim configuration framework for cross-platform development, supporting Lua scripts, plugin management, and theme customization.  

## Features  
- **Modular Configuration**: Organized into `config` and `plugins` directories for easy maintenance.  
- **Plugin Management**: Pre-configured plugin integration with `.lua` and `.toml` configuration files.  
- **Theme Support**: Customizable color schemes and UI themes via `.lua` scripts.  
- **Cross-Platform Compatibility**: Works seamlessly across macOS, Linux, and Windows.  
- **Git Integration**: Includes `.git` metadata for version control tracking.  

## Installation  
### For Neovim Users  
1. Clone the repository:  
   ```bash  
   git clone https://github.com/your-username/NVim-Config.git  
   ```  
2. Place the cloned directory in your Neovim configuration path (e.g., `~/.config/nvim/`).  

### For Xcode Projects (if applicable)  
1. **Deployment Targets**: macOS 10.15+, iOS 13.0+  
2. **Xcode Version**: 14.0+  
3. **Swift Version**: Swift 5.9+  
4. **Dependency Management**:  
   - **CocoaPods**:  
     ```bash  
     pod install  
     ```  
   - **Swift Package Manager**:  
     ```bash  
     swift package resolve  
     ```  
   - **Carthage**:  
     ```bash  
     carthage update  
     ```  
5. **Build Instructions**:  
   - Open `YourProject.xcodeproj` in Xcode.  
   - Select the appropriate deployment target.  
   - Build and run the project.  

## Usage Examples  
1. **Customize Keybindings**:  
   ```lua  
   vim.keymap.set("n", "<leader>sv", ":w<CR>:q<CR>")  
   ```  
2. **Enable Plugins**:  
   Add plugins to the `plugins` directory and configure them in `config/plugins.lua`.  
3. **Switch Themes**:  
   Update `config/theme.lua` with your preferred color scheme.

## Dependencies & Requirements  
- **Neovim**: v0.12+ (for Lua-based configurations).  
- **Git**: Required for version control.  
- **CocoaPods/Swift Package Manager/Carthage**: For iOS/macOS integration (if applicable).  
- **Swift**: 5.9+ (for Xcode-based projects).

## 安装说明  
### Neovim 配置部署  
1. 克隆仓库：  
   ```bash  
   git clone https://github.com/your-username/NVim-Config.git ~/.config/nvim  
   ```  
2. 链接配置文件：  
   ```bash  
   ln -s ~/.config/nvim/init.lua ~/.config/nvim/init.lua  
   ```  

### Xcode 项目集成（iOS/macOS）  
1. 部署目标：iOS 15.0 / macOS 12.0  
2. Xcode 版本要求：14.0+  
3. Swift 版本：5.9+  
4. CocoaPods 集成：  
   ```bash  
   cd YourProjectFolder  
   pod init  
   pod 'NVim-Config', :path => '../NVim-Config'  
   ```  
5. SPM 集成：  
   - 在 Xcode 中选择 `File > Add Packages...`  
   - 输入仓库地址并添加  
6. Carthage 集成：  
   ```bash  
   cd YourProjectFolder  
   echo 'github "your-username/NVim-Config" ~> 1.0' > Cartfile  
   carthage update --platform iOS  
   ```
